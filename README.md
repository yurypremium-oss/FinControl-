# FinControl

Aplicativo de gerenciamento financeiro desenvolvido em Flutter.

## Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Dashboard financeiro
- Relatórios
- Exportação em PDF
- Exportação em Excel
- Controle de cartões
- Metas financeiras

## Tecnologias

- Flutter
- Dart
- Supabase
- PostgreSQL
- FL Chart
